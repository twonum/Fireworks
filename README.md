# Atopes - Fireworks Project ðŸŽ†

**Atopes** is a visually stunning and interactive Next.js project designed to celebrate special moments with a dynamic fireworks animation. The website features a beautiful, starry background with vibrant fireworks triggered by user interaction. Perfect for a romantic gesture or a festive celebration, this project uses advanced animations, sound effects, and seamless user experience for a magical, immersive feel.

## Features:
- Interactive fireworks animation triggered by button click
- Vibrant starry background that enhances the visual experience
- Customizable fireworks settings (launch speed, frequency, explosion effects)
- Smooth animations with Framer Motion for smooth transitions
- Sound effects for a more immersive experience using Howler.js

## Tech Stack:
- **Frontend**: Next.js, React, Framer Motion, Tailwind CSS
- **Animation & Effects**: Three.js (via @react-three/fiber), Fireworks.js, Howler.js
- **Styling**: Tailwind CSS for responsive design

## How to Run Locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/twonum/Fireworks.git
   ```

2. Navigate into the project directory:
   ```bash
   cd atopes-fireworks-project
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Visit `http://localhost:3000` in your browser.

## Deployment:
This project is deployed on [Vercel](https://vercel.com/) for seamless serverless hosting.
